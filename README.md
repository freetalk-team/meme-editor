# Meme Editor

Create memes in browser

## Try it

```
chromium demo.html
```

## Install

```
npm install
```

## Start web server

Start web server. Default port is **3013**

```
npm start
```

or use other port

```
PORT=5000 npm start
```

[Open in browser](http://127.0.0.1:3013)

## Build

1. Install **webpack** globally if not installed already

```
npm install -g webpack
```

2. Build javascript code

```
npm run build
```
